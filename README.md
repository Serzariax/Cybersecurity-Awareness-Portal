
# Cybersecurity Awareness Portal

A comprehensive web application designed to enhance cybersecurity awareness and provide essential training resources for organizations and individuals.

## Overview

The Cybersecurity Awareness Portal is a React-based web application that offers interactive training modules, incident reporting tools, and real-time security guidance. Built with modern web technologies, it provides an engaging platform for learning about cybersecurity best practices and staying informed about current threats.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Serzariax/Cybersecurity-Awareness-Portal.git
   ```

2. Install dependencies:
   ```bash
   cd Cybersecurity-Awareness-Portal
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`
